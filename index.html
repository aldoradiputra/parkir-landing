Tiket & Kartu = Ribet

Tiket kertas hilang, kartu e-money (Flazz, e-money Mandiri) harus di-tap manual, saldo kurang harus isi ulang dulu. Semua ini seharusnya tidak perlu ada — plat nomor Anda sudah cukup.

Solusi

Satu Sistem.
Semua Teratasi.

SupaPark menggantikan seluruh infrastruktur parkir konvensional dengan teknologi AI yang terjangkau, transparan, dan bisa jalan tanpa operator.

ALPR Otomatis

Kamera AI baca plat nomor dalam 0.5-2 detik. Masuk dan keluar tanpa tiket, tanpa kartu. Plat Anda = identitas Anda.

Full QRIS — Standar Nasional Terbaik

Bayar prepaid via PWA sebelum ke exit, atau scan QRIS / tap NFC QRIS di layar exit. Semua e-wallet: GoPay, OVO, Dana, ShopeePay, LinkAja. Satu standar, tanpa ribet kartu e-money.

Tanpa Operator

Palang buka otomatis, pembayaran digital, notifikasi real-time. Hemat Rp 5-10 juta/bulan biaya operator per shift.

Offline-First

Internet mati? Sistem tetap jalan. Data tersimpan lokal, palang tetap buka, tagihan dikirim saat online. Zero downtime.

Link Plat ke HP — Next Gen Parking

Scan QR di area parkir → link plat nomor ke nomor HP via PWA → terima notifikasi masuk/keluar → prepay sebelum ke exit. Parkir generasi berikutnya, langsung dari browser.

Dashboard Real-time

Monitor semua lokasi, pendapatan, okupansi dari satu dashboard. Bilingual EN/ID. Akses dari HP, tablet, atau laptop.

Member Tanpa Kartu

Daftar sekali pakai plat nomor, berlaku di semua lokasi SupaPark. Tanpa kartu fisik, tanpa biaya cetak.

Hardware Fleksibel

Pakai palang & kamera existing ATAU kit baru dari kami. Tidak ada vendor lock-in. Komponen tersedia di Tokopedia.

Arsitektur Sistem

Edge Device

  • Raspberry Pi 4/5 per jalur
  • ALPR engine (YOLOv8 + EasyOCR)
  • Kontrol palang via GPIO relay
  • SQLite penyimpanan lokal
  • PWA embedded (no server)

Cloud Server

  • PostgreSQL database
  • REST API + WebSocket
  • Midtrans payment gateway
  • WhatsApp via Fonnte
  • Multi-tenant dashboard

Interfaces

  • Dashboard operator (Next.js)
  • Layar booth touchscreen
  • Tablet petugas
  • PWA pengunjung
  • WhatsApp notifikasi
